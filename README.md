# OFTodo

To do app exercice with Webpack React Redux json-server and Axios

## Installation

git clone https://github.com/castorzombie/oftodo.git
cd oftodo
npm install
npm install -g json-server

## Usage
json-server --watch todo.json --port 3000
npm start
npm run-script build
